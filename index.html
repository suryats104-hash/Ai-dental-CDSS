How to get your own key
1. Go to console.groq.com and create a free account.
2. Click API Keys in the left sidebar.
3. Click Create API Key, give it a name and copy it.
4. Paste it above and click Save Key.
⚠️ Your key is stored only in this browser's local storage — it is never transmitted anywhere except directly to Groq's API. Clear it anytime using the button above.
ℹ About This App
Dental CDSS is an AI-powered Clinical Decision Support System for dental house surgeons in India.

Powered by LLaMA 3.3 70B via Groq, it provides diagnosis, investigations, and weight-based prescription recommendations based on the patient's clinical profile.

All case data is stored locally in your browser using IndexedDB and never leaves your device.
