1 API Configuration
Your API key is stored locally in your browser and never sent to any server except Groq API. Leave blank to use the default key.
2 How to Get Your Groq API Key
1
Visit Groq Cloud Console
Go to console.groq.com and sign up or log in to your account.
2
Navigate to API Keys
Click on "API Keys" in the left sidebar menu.
3
Create New Key
Click "Create API Key", give it a name (e.g., "Dental CDSS"), and optionally set expiration/permissions.
4
Copy Your Key
Copy the generated key (starts with gsk_) and paste it into the API Key field above.
⚠️ Save your key somewhere safe — you can only see it once when created!
ℹ️ About Groq API

Groq provides free tier access to powerful AI models including LLaMA 3.3 70B for text analysis and LLaMA 3.2 90B Vision for image analysis. The free tier includes 30 requests per minute, which is sufficient for most dental practices.

4 Data & Privacy
🔒
Local Storage

Your API key and case history are stored locally in your browser using IndexedDB. Nothing is sent to our servers.

☁️
API Communication

Patient data is sent only to Groq API servers for analysis. Groq's privacy policy applies to data processed through their API.

🗑️
Data Deletion

Clear your browser data to delete all stored information. Individual cases can be deleted from the History tab.
